Réalisation Power BI - CasatramwayCover

For data visualization, I created a comprehensive Power BI dashboard to analyze the 1,369 participants of the CasatramwayCover event. It displays participant information and interactive charts showing tramway usage (regular, occasional, or non-user) and registration distribution by platform (Facebook, Instagram).

Before development, I designed the data model using class and schema diagrams, based on a star schema with a fact table (user journeys) and multiple dimension tables (users, platforms, dates, usage). Data was imported from Excel into Power BI for complete multidimensional analysis.
